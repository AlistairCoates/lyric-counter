## lyric-counter

# Use pip to install (I suggest using a virtualenv):

```
pip install -e .
```

# Requirements:
python3
```
pip install asyncio
pip install aiohttp
pip install matplotlib
pip install numpy
```

# Use (CLI)
Clone from repo

For a single artist
```
lyric-counter <artist>
```

For multiple artists
```
lyric-counter <artist1>,<artist2>,<artist3>
```
